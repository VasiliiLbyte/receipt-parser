# Интеграция с 1С через внешнюю обработку

Эта папка содержит исходник внешней обработки для 1С 8.3+:
- `ReceiptParserLoader.bsl` — модуль обработки на встроенном языке 1С.

## Установка (5-7 шагов)

1. Откройте базу 1С (Бухгалтерия 3.0, УНФ или УТ 11) в режиме предприятия.
2. Перейдите в меню `Файл -> Открыть` (или `Администрирование -> Дополнительные отчеты и обработки`).
3. Создайте/подключите внешнюю обработку и вставьте код из `ReceiptParserLoader.bsl` в модуль формы обработки.
4. Создайте на форме реквизиты: `СерверURL`, `APIКлюч`, `ПользовательID` и кнопку `Загрузить`.
5. Для кнопки назначьте обработчик `КнопкаЗагрузитьНажатие`.
6. Заполните параметры:
   - `СерверURL` (например, `http://localhost:8000`)
   - `APIКлюч` (если включен `API_KEY` в backend)
   - `ПользовательID` (ID пользователя, чьи чеки нужно загрузить)
7. Нажмите `Загрузить` и проверьте созданные документы в 1С.

## Важно

- Файл `.epf` здесь не формируется: компиляцию делает разработчик в Конфигураторе 1С.
- Тип документа в примере — `АвансовыйОтчет`; при необходимости адаптируйте под вашу конфигурацию.
