# Блок А: Настройка конфигурации 1С:Предприятие 8.3 с нуля

## Цель
Создать минимальную конфигурацию для приёма чеков из Receipt Parser API.
Вся работа выполняется в **Конфигураторе** 1С:Предприятие 8.3.

---

## А0. Создание базы данных

1. Запустить `1cestart.exe` → **Добавить** → **Создание новой информационной базы**
2. **Создание информационной базы без конфигурации**
3. Указать имя: `ReceiptParser`, путь к папке базы
4. Открыть базу в режиме **Конфигуратор**

---

## А1. Справочник «Контрагенты»

**Конфигуратор → Конфигурация → Открыть конфигурацию → ПКМ на «Справочники» → Добавить**

| Поле | Значение |
|------|----------|
| Имя | `Контрагенты` |
| Синоним | `Контрагенты` |

**Реквизиты** (вкладка «Данные» → «Реквизиты» → кнопка «Добавить»):

| Имя | Тип | Длина |
|-----|-----|-------|
| `ИНН` | Строка | 12 |
| `КПП` | Строка | 9 |
| `ЮридическийАдрес` | Строка | 200 |

> ⚙️ **Промт для Cursor:**
> ```
> В конфигурации 1С:Предприятие 8.3 создай справочник "Контрагенты".
> Добавь реквизиты: ИНН (Строка, длина 12), КПП (Строка, длина 9),
> ЮридическийАдрес (Строка, длина 200).
> Стандартный реквизит "Наименование" оставить как есть (Строка, 150).
> ```

## А4. Документ «АвансовыйОтчет»

**ПКМ на «Документы» → Добавить**

| Поле | Значение |
|------|----------|
| Имя | `АвансовыйОтчет` |
| Синоним | `Авансовый отчет` |
| Нумерация | Общая, длина номера 7 |

### Реквизиты шапки (вкладка «Данные» → «Реквизиты»):

| Имя | Тип | Параметры |
|-----|-----|-----------|
| `Контрагент` | СправочникСсылка.Контрагенты | — |
| `СуммаДокумента` | Число | Длина 15, точность 2 |
| `НомерЧека` | Строка | Длина 20 |
| `ДатаЧека` | Дата | Дата |
| `СуммаНДС` | Число | Длина 15, точность 2 |

### Табличная часть «Товары»:

1. Вкладка «Данные» → «Табличные части» → **Добавить** → Имя `Товары`
2. Выбрать «Товары» → «Реквизиты табличной части» → добавить:

| Имя | Тип | Параметры |
|-----|-----|-----------|
| `Номенклатура` | СправочникСсылка.Номенклатура | — |
| `Количество` | Число | Длина 10, точность 3 |
| `Цена` | Число | Длина 15, точность 2 |
| `Сумма` | Число | Длина 15, точность 2 |
| `СтавкаНДС` | Строка | Длина 10 |
| `СуммаНДС` | Число | Длина 15, точность 2 |

> ⚙️ **Промт для Cursor:**
> ```
> В конфигурации 1С:Предприятие 8.3 создай документ "АвансовыйОтчет".
> Реквизиты шапки:
> - Контрагент (тип: СправочникСсылка.Контрагенты)
> - СуммаДокумента (Число, длина 15, точность 2)
> - НомерЧека (Строка, длина 20)
> - ДатаЧека (Дата)
> - СуммаНДС (Число, длина 15, точность 2)
> Табличная часть "Товары" с колонками:
> - Номенклатура (тип: СправочникСсылка.Номенклатура)
> - Количество (Число, длина 10, точность 3)
> - Цена (Число, длина 15, точность 2)
> - Сумма (Число, длина 15, точность 2)
> - СтавкаНДС (Строка, длина 10)
> - СуммаНДС (Число, длина 15, точность 2)
> ```

## А5. Роль «ОсновнаяРоль»

**ПКМ на «Роли» → Добавить**

| Поле | Значение |
|------|----------|
| Имя | `ОсновнаяРоль` |
| Синоним | `Основная роль` |

**Права:**
1. Открыть роль → вкладка «Права»
2. Нажать кнопку **«Установить все права»** (или вручную проставить галки на все объекты)
3. Убедиться что для справочников `Контрагенты`, `Номенклатура` и документа `АвансовыйОтчет` стоят права:
   - Чтение ✅
   - Добавление ✅
   - Изменение ✅
   - Удаление ✅
   - Проведение ✅ (для документа)

> ⚙️ **Промт для Cursor:**
> ```
> В конфигурации 1С:Предприятие 8.3 создай роль "ОсновнаяРоль"
> с полными правами на все объекты конфигурации:
> справочники Контрагенты и Номенклатура,
> перечисление СтавкиНДС,
> документ АвансовыйОтчет.
> ```

## А6. Пользователь «Бухгалтер»

**Конфигуратор → Администрирование → Пользователи → Добавить**

| Поле | Значение |
|------|----------|
| Имя | `Бухгалтер` |
| Полное имя | `Бухгалтер` |
| Пароль | (пустой) |
| Роль | `ОсновнаяРоль` (поставить галку) |
| Основной интерфейс | Обычное приложение или оставить пусто |

---

## А7. Финальные шаги

1. **Конфигуратор → Конфигурация → Обновить конфигурацию базы данных** (F7)
2. Подтвердить обновление → **Принять**
3. Запустить **1С:Предприятие** (Ctrl+F5) с пользователем `Бухгалтер`
4. Убедиться что видны меню «Справочники» и «Документы»
